# tweepy_auto_follow
Twitterの自動フォロースクリプトです。
気になる単語を指定するだけで、関連するユーザを自動フォローします。
 
# Requirement

* Python 3.X.X
 
# Installation
 
```bash
pip install requirement.txt
```
 
# Usage
  
```bash
python autofollow {search_words}
```
 
# Note
※ スパムと間違えられて、アカウントが凍結する可能性があるのでご注意ください。
